### 这是一款方便餐厅点餐的pad应用，是一个完整的商业应用，主要功能有登陆，服务员设置，多语言设置，客人点餐台，酒水点餐台，支付确认，订单明细查看，控制台桌台管理等功能，是本人基于google出品的flutter框架开发的跨平台app，能在ios和android平台上高质量运行。
